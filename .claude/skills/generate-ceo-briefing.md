# Skill: generate-ceo-briefing

## Description
Create weekly CEO Briefing document summarizing business performance, bottlenecks, and suggestions.

## Phase
Gold (Autonomous Employee)

## Trigger
- Sunday night scheduled task
- Manual invocation
- End of business week

## Instructions

You are generating the CEO Briefing. Follow these steps:

### 1. Gather Data Sources
Read from these locations:
- `/Business_Goals.md` - Targets and objectives
- `/Accounting/` - Financial data (Odoo summaries)
- `/Done/` - Completed tasks (last 7 days)
- `/Logs/` - All actions (last 7 days)
- `/Pending_Approval/` - Outstanding items

### 2. Calculate Metrics

#### Revenue
- Sum all income from Accounting data
- Compare to monthly/weekly targets
- Calculate percentage progress

#### Task Analysis
- Count completed tasks
- Identify delayed tasks (expected vs actual)
- Calculate completion rate

#### Cost Analysis
- Identify recurring subscriptions
- Flag unused subscriptions (no activity 30+ days)
- Calculate total software costs

### 3. Generate Briefing
Create `/Briefings/YYYY-MM-DD_Monday_Briefing.md`:

```markdown
---
generated: [ISO_TIMESTAMP]
period: [START_DATE] to [END_DATE]
---

# Monday Morning CEO Briefing

## Executive Summary
[2-3 sentence overview of the week]

## Revenue
- **This Week**: $[amount]
- **MTD**: $[amount] ([%] of $[target] target)
- **Trend**: [On track | Behind | Ahead]

## Completed Tasks
- [x] [Task 1 description]
- [x] [Task 2 description]
- [x] [Task 3 description]

## Bottlenecks
| Task | Expected | Actual | Delay |
|------|----------|--------|-------|
| [Task] | [X days] | [Y days] | [+Z days] |

## Proactive Suggestions

### Cost Optimization
- **[Service]**: [Issue]. Cost: $[amount]/month.
  - [ACTION] [Suggestion]? Move to /Pending_Approval

### Process Improvements
- [Observation about workflow]
- [Suggestion for improvement]

## Upcoming Deadlines
- [Project/Task]: [Date] ([X days])
- [Project/Task]: [Date] ([X days])

## Risks & Alerts
- [Any concerning patterns or issues]

---
*Generated by AI Employee v1.0*
```

### 4. Update Dashboard
Add briefing link to Recent Activity.

### 5. Archive
Previous briefings remain in /Briefings for historical reference.

## Success Criteria
- Briefing generated with all sections
- Accurate financial data
- Actionable suggestions included
- Dashboard updated
